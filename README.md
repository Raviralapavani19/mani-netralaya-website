Mani Netralaya Website

This is a responsive website developed for Mani Netralaya, an eye care clinic located in Manneguda, Hyderabad.

Features

- Responsive design (mobile & desktop)
- Clean and user-friendly UI
- WhatsApp integration for appointments
- Services section for treatments

Technologies Used

- HTML
- CSS
- JavaScript

Live Website

https://maninetralaya.in

Project Description

This project was built to help a real clinic establish an online presence and improve accessibility for patients.
